About
MST Search is a web system which can crawl through websites and have it's information indexed so that the user can search throughout it.

This system was made as part of a dissertation project while within the University of Sunderland. The client wanted a system which allowed them to submit their own personal websites and have the ability to search throughout them all. It indexes all of the text that is listed within the website, as well as any images that may appear. The system was built for one sole user so it is not possible for demonstration. However, screenshows will be provided of the system below.

Disclaimer: As this was a project created both for university and accenture, it is currently unknown if this system is actually being used by the user.
